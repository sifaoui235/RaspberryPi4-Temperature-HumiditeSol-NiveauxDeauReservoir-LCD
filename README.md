# RaspberryPi4-Temperature-Humidit-Sol-NiveauxDeauReservoir-LCD
Assemblage des mini projets de mesure de temperature de l'humidité de sol et niveau d'eau du réservoir avec code d'ajouter l'alerte par email 

# Objectif du projet
- Mesurer la température ambiante, l’humidité du sol et le niveau d’eau dans un réservoir.
- Afficher toutes les mesures en temps réel sur un écran LCD.
- Envoyer des alertes par email si les seuils critiques sont dépassés.
- Fournir un système complet pour l’irrigation et du suivi environnemental.

# Contenu
- RaspberryPi4-Temperature-Humidite-Sol-NiveauEauReservoir.pdf – Documentation complète du projet.
- code-alert-email.txt – Pour l’alerte email.

# Technologies
- Raspberry Pi 4
- Capteur DHT22 (température et humidité)
- Capteur d’humidité du sol
- Capteur de niveau d’eau
- Écran LCD
- 2 pompes 5v
- 2 relais
- batterie 9v
- Email pour alertes
- Langage Python
